## 🔑 配置真实ChatGPT API

### 步骤1：获取OpenAI API密钥
1. 访问 [OpenAI平台](https://platform.openai.com/api-keys)
2. 注册账户并创建API密钥
3. 复制您的API密钥

### 步骤2：配置环境变量
1. 在项目根目录找到 `.env.local` 文件
2. 添加以下配置：
```bash
OPENAI_API_KEY=your_openai_api_key_here
```

### 步骤3：重启服务器
```bash
npm run dev
# 或
pnpm dev
```

## 📞 故障排除

### 常见问题：
1. **聊天按钮不显示**：检查浏览器控制台错误
2. **API调用失败**：验证API密钥配置
3. **回复很慢**：检查网络连接或API额度
